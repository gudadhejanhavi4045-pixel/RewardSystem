# Reward-Based Question & Answer System

## Project Description
This project is a Reward-Based Question & Answer System developed using PHP, MySQL, HTML, and CSS. It allows users to ask questions, answer questions, earn reward points, transfer points, and view a leaderboard.

## Features
- User Registration and Login
- Ask Questions
- Answer Questions
- Reward Points (+5 for every answer)
- Bonus +5 points after 5 upvotes
- Upvote and Downvote System
- Transfer Reward Points
- User Profile
- Leaderboard

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## Installation
1. Install XAMPP.
2. Copy the project folder into `htdocs`.
3. Start Apache and MySQL.
4. Create a database named `reward_system`.
5. Import the database (if the SQL file is available).
6. Open:
   http://localhost/RewardSystem/pages/home.php
